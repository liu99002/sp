## 針對 C0 語言的系統程式

由於 nand2tetris 的 Part 2 採用類似 Java 的 Jack 語言，但《系統程式》通常採用 C 語言，因此也可以編譯器的語言改為類似 C 的語言 （例如 C0 語言）。
